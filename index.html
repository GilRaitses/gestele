Steps to Get Started

  1. Clone the repository: 
    git clone https://github.com/gilraitses/gestele.git
  2. Navigate to the project directory: 
    cd gestele
  3. Set up Python dependencies: 
    pip install -r requirements.txt
  4. Run the app: 
    streamlit run app.py
    This will launch the app in your default browser.

How to Use the Interface

Once the app is running, follow these steps:

  1. Enter a folder path: Paste the path to the folder containing your codebase. You can find this path on your system by right-clicking the folder and copying its full path.
  2. Generate a preview: Click the "Generate Preview" button to view the structure of your codebase. Hermes will create a Table of Contents and display the contents of your files.
  3. Apply the 'Cutieness Filter': Click the "Apply Cutieness" button to add playful styles, colors, and emojis to your codebase.
  4. Export your work: Use the "Download Markdown" or "Download PDF" buttons to save your beautifully formatted output. Choose Markdown for editing or PDF for sharing.

Why Use Gestele?

  • Turn raw code into organized, visually appealing documents.
  • Export your work in multiple formats (HTML, Markdown, PDF).
  • Enjoy a playful experience with Hermes' guidance.
